🚀 AI Consulting Assistant (100% Free & Open Source)
📌 Overview

This project is a full-stack AI-powered consulting assistant built entirely using free and open-source tools. It helps analyze requirements, recommend architectures, and assist with code migration using locally hosted LLMs.

🧠 Features
🔍 Requirements Analyzer
🧱 Architecture Recommender
🔄 Code Migration Tool
🤖 AI Copilot (Local LLM)
📄 Report Generator
🧰 Tech Stack
Backend: FastAPI (Python)
Frontend: Streamlit
AI: Ollama (Mistral / LLaMA3)
Database: SQLite
Vector DB: FAISS
Deployment: Render / Railway / Local
